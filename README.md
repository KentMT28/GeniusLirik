# GeniusLirik
GeniusLyrics: Generator lirik AI dengan Bi-LSTM &amp; Word2Vec. Buat lirik koheren dalam bahasa Indonesia &amp; Inggris dari dataset Genius.com (8.925 Inggris, 3.396 Indonesia). Fitur: seed text, genre (Pop, Rock, dll.), jumlah kata, temperatur. Web app Flask. Bantu musisi atasi writer's block!
